# 🔐 Secrets Keeper

A full-stack web app that allows users to **register**, **log in**, and **share their secrets anonymously**. This project demonstrates secure user authentication using **Passport.js**, password hashing with **bcrypt**, and session handling with **Express Sessions**.

---

## 🚀 Features

- 🔒 User authentication with **Passport.js** (local strategy)
- 🧂 Passwords are securely hashed using **bcrypt**
- 🗝️ Session-based login and logout system
- 🧾 PostgreSQL database for user storage
- 🎨 Simple and clean UI using **EJS** templating

---

## 🛠️ Tech Stack

- **Node.js** & **Express.js** – Backend and routing
- **PostgreSQL** – User database
- **Passport.js** – Authentication middleware
- **bcrypt** – Secure password hashing
- **express-session** – Session handling
- **EJS** – Templating engine for frontend
